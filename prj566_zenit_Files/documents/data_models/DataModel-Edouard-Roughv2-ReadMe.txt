The data model does not show any arrows due to readability 
issues. 

How to view the Foreign Key Info in Visio:

1)Select an"Entity", the "Database Properties"(*) will be 
displayed.

	(*) If the properties are not displayed 
	    Double Click the "Entity"

2)Check the "Notes" Category for the foreign keys of that 
table.

(**) This data model was made using Visio 2010. Not tested 
on older versions of Visio.
